Política de Privacidad

Esta aplicación utiliza la API de WhatsApp Cloud proporcionada por Meta para facilitar la comunicación entre empresas y usuarios.

Recopilación de datos

No almacenamos datos personales directamente. Los mensajes y datos son gestionados por Meta y están sujetos a sus políticas.

Uso de la información

La información enviada a través del servicio se usa exclusivamente para facilitar la comunicación. No compartimos ni vendemos datos.

Responsabilidad

Este servicio no es propiedad de Meta ni tiene afiliación directa. Solo usamos su API pública de forma legal y segura.
